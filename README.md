## Introduction

This is a ChatRoom HTML5 App based on Tornado, Redis, Bootstrap3

# Environment 
```
OS: MacOS X EI  
Redis: 3.0  
Python: 2.7.10  
```

# Usage
```
1. make sure redis server is running
2. clone this repo

$ virtualenv chat_env   
$ source chat_env/bin/activate  
$ (chat_env)# pip install -r requirements.txt  

$ modify "config.py"
$ python chat.py  
```
